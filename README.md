# Student Performance Prediction

## Overview

This project aims to predict student academic performance using Machine Learning techniques.

The goal is to analyze various academic, demographic, and study-related factors and build predictive models that estimate student performance.

---

## Problem Statement

Educational institutions often need to identify students who may require additional academic support.

This project explores whether machine learning models can predict student performance based on factors such as:

* Study time
* Attendance
* Previous academic records
* Family background
* Extracurricular activities
